TensorFlow - tf.random_uniform - Explicación y ejemplos¶

Todos los archivos utilizados en este documento se pueden encontrar en este repositorio de GitHub. Se recomienda utilizar estos recursos para comprender mejor los temas explicados.

La finalidad de este documento es explicar y ejemplificar los usos de la función "random_normal", provista en la librería de TensorFlow, para Python.

Dicha función es utilizada para generar un tensor con valores aleatorios, que siguen una distribución normal, con valores de media y desviación estandar conocidos.

La distribución de los valores generados, se podrá ver con mayor detalle en los ejemplos del final del documento.

Argumentos¶

tf.random_normal(shape, mean=0.0, stddev=1.0, dtype=tf.float32, seed=None, name=None)

Los argumentos de la función son:

  • shape: Las dimensiones del tensor de salida. Se puede pasar a la función como un tensor 1-D de números entros, o como una lista de Python de números enteros. Este es el único argumento requerido de la función, los demás son opcionales.

  • mean [ Valor predeterminado: mean = 0.0 ] : La media de la distribución normal, que seguirán los números generados. Se puede pasar a la función como un tensor 0-D, o un valor de Python, de tipo dtype.

  • stddev [ Valor predeterminado: stddev = 1.0 ] : Es la desviación estándar de la distribución normal, que seguirán los números generados. Se puede pasar a la función como un tensor 0-D, o un valor de Python, de tipo dtype.

  • dtype [ Valor predeterminado: dtype = tf.float32 ] : El tipo de valores que quiero generar. Puede ser float32, float64, int32, o int64.

  • seed [ Valor predeterminado: seed = None ] : Un número entero de Python. Es el generador de numeros aleatorios. Se utiliza para que sucesivas ejecuciones del programa generen siempre los mismos valores aleatorios, en el mismo orden. Es util cuando se esta depurando el programa, pero no se recomienda utilizarlo cuando el programa esta terminado.

  • name [ Valor predeterminado: name = None ] : El nombre de la operación. Lo utiliza TensorFlow para identificar distintas operaciones dentro del mismo programa. Se pasa a la función como una cadena de texto de Python.

Ejemplos de uso¶

In [1]:
import tensorflow as tf
import matplotlib.pyplot as plt
import numpy as np
%matplotlib inline

El ejemplo de uso más básico es el de utilizar la funcion con todos los parametros con valores predeterminados. De esta forma estamos creando un tensor de números aleatorios, cuyos valores siguen una distribución normal con media 0.0, desviación estandar 1.0, y son de tipo float32. Se ejemplificará además, el uso del atributo name.

In [2]:
# Creación de un tensor de 2 x 5, con una lista de Python de números enteros
tensor_normal_1 = tf.random_normal([2, 5])

# Creación de un tensor de 5 x 3, con un tensor de números enteros
tensor = tf.constant([5, 3])
tensor_normal_2 = tf.random_normal(tensor, name="nombre_ejemplo")

# Visualización de los tensores creados
print("Tensor 1: {}".format(tensor_normal_1))
print("Tensor 2: {}".format(tensor_normal_2))

Tensor 1: Tensor("random_normal:0", shape=(2, 5), dtype=float32)
Tensor 2: Tensor("nombre_ejemplo:0", shape=(5, 3), dtype=float32)

Como se puede ver, los objetos creados con la función tf.random_normal, son tensores de tipo float32, y dimensiones (2, 5) y (5, 3), respectivamente. Además se puede ver el nombre de cada tensor. Cuando no se especifico el atributo name, el tensor obtuvo su nombre de la función que lo creó. Cuando se especifica un nombre, el tensor lo adquiere como identidificación.

Los tensores actualmente no contienen ningun valor, para generarlos, se deben correr los mismos en una Session de TensorFlow.

In [3]:
# Generación de los valores aleatorios
with tf.Session() as sess:
    tensor_normal_1 = sess.run(tensor_normal_1)
    tensor_normal_2 = sess.run(tensor_normal_2)

# Impresión de los resultados
print("Tensor con valores aleatorios N° 1: (shape={})".format(tensor_normal_1.shape))
print(tensor_normal_1)
print()
print("Tensor con valores aleatorios N° 2: (shape={})".format(tensor_normal_2.shape))
print(tensor_normal_2)

Como se puede observar, al no haber especificado valores para la media y la desviación estandar, la función utilizó los valores predeterminados de mean y stddev. Se puede ver cómo todos los valores generados están ubicados a ambos lados de la media. Sin embargo para visualizar efectivamente la distribición uniforme que siguen los mismos, es necesario generar más valores, para que la cantidad sea significativa.

Observando la distribución de los valores¶

Para comprender mejor la distribucion normal que siguen los valores generados, se graficarán en un histograma de 10 clases, los valores generados en 2 tensores con 100 y 1000 números, respectivamente. Además se especificarán los atributos media y stddev, en uno de los tensores, para ejemplificar su uso.

In [7]:
# Creación de ambos tensores
tensor_100 = tf.random_normal([100])
tensor_1000 = tf.random_normal([1000], mean=15, stddev=4)

# Generación de los valores aleatorios
with tf.Session() as sess:
    tensor_100 = sess.run(tensor_100)
    tensor_1000 = sess.run(tensor_1000)

# Importación de función de ayuda
from histogram_plots import histograms_x2
histograms_x2(tensor_100, tensor_1000)

Como se puede ver en los gráficos, los valores generados, siguen una distribución normal, con centro en mean. Al aumentar la cantidad de valores generados nos acercamos todavía más a la distribución normal teórica.

Las figuras muestran además, que el promedio de los valores aleatorios creados, está muy cerca del valor de la media. Esto es otra muestra del comportamiento normal de la población de números generados.
