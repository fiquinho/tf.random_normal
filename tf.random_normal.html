Todos los archivos utilizados en este documento se pueden encontrar en este repositorio de GitHub. Se recomienda utilizar estos recursos para comprender mejor los temas explicados.

La finalidad de este documento es explicar y ejemplificar los usos de la funcin "random_normal", provista en la librera de TensorFlow, para Python.

Dicha funcin es utilizada para generar un tensor con valores aleatorios, que siguen una distribucin normal, con valores de media y desviacin estandar conocidos.

La distribucin de los valores generados, se podr ver con mayor detalle en los ejemplos del final del documento.

Argumentos¶

tf.random_normal(shape, mean=0.0, stddev=1.0, dtype=tf.float32, seed=None, name=None)

Los argumentos de la funcin son:

  • shape: Las dimensiones del tensor de salida. Se puede pasar a la funcin como un tensor 1-D de nmeros entros, o como una lista de Python de nmeros enteros. Este es el nico argumento requerido de la funcin, los dems son opcionales.

  • mean [ Valor predeterminado: mean = 0.0 ] : La media de la distribucin normal, que seguirn los nmeros generados. Se puede pasar a la funcin como un tensor 0-D, o un valor de Python, de tipo dtype.

  • stddev [ Valor predeterminado: stddev = 1.0 ] : Es la desviacin estndar de la distribucin normal, que seguirn los nmeros generados. Se puede pasar a la funcin como un tensor 0-D, o un valor de Python, de tipo dtype.

  • dtype [ Valor predeterminado: dtype = tf.float32 ] : El tipo de valores que quiero generar. Puede ser float32, float64, int32, o int64.

  • seed [ Valor predeterminado: seed = None ] : Un nmero entero de Python. Es el generador de numeros aleatorios. Se utiliza para que sucesivas ejecuciones del programa generen siempre los mismos valores aleatorios, en el mismo orden. Es util cuando se esta depurando el programa, pero no se recomienda utilizarlo cuando el programa esta terminado.

  • name [ Valor predeterminado: name = None ] : El nombre de la operacin. Lo utiliza TensorFlow para identificar distintas operaciones dentro del mismo programa. Se pasa a la funcin como una cadena de texto de Python.

El ejemplo de uso ms bsico es el de utilizar la funcion con todos los parametros con valores predeterminados. De esta forma estamos creando un tensor de nmeros aleatorios, cuyos valores siguen una distribucin normal con media 0.0, desviacin estandar 1.0, y son de tipo float32. Se ejemplificar adems, el uso del atributo name.

# Creacin de un tensor de 2 x 5, con una lista de Python de nmeros enteros

# Creacin de un tensor de 5 x 3, con un tensor de nmeros enteros

tensor_normal_2 = tf.random_normal(tensor, name="nombre_ejemplo")

# Visualizacin de los tensores creados

Tensor 1: Tensor("random_normal:0", shape=(2, 5), dtype=float32)

Como se puede ver, los objetos creados con la funcin tf.random_normal, son tensores de tipo float32, y dimensiones (2, 5) y (5, 3), respectivamente. Adems se puede ver el nombre de cada tensor. Cuando no se especifico el atributo name, el tensor obtuvo su nombre de la funcin que lo cre. Cuando se especifica un nombre, el tensor lo adquiere como identidificacin.

Los tensores actualmente no contienen ningun valor, para generarlos, se deben correr los mismos en una Session de TensorFlow.
